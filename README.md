# Expense Tracker CLI

A simple command-line tool to track your expenses. Built with Node.js and Commander

# Installation
```
git clone https://github.com/sectoruu/cli-expense-tracker.git  
cd ExpenseTracker  
npm install
```  

# Usage
```
expense-tracker <command> [options]

Options:  
  -V, --version      output the version number  
  -h, --help         display help for command  

Commands:  
  add [options]      Add new expense  
  delete [options]   Delete expense  
  summary [options]  Summary expenses  
  list               List all expenses  
  help [command]     display help for command
```
